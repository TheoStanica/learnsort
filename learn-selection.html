Sortarea in programare se refera in general la aranjarea unui tablou de elemente astfel incat elementele lui sunt asezate intr-o anumita ordine (crescatoare sau descrescatoare)

De exemplu, dupa ce vom rearanja vectorul v = [5, 7, 12, 1, 3, 8, 10, 9], vom obtine v = [1, 3, 5, 7, 8, 9, 10, 12]

Sortarea este o operatiune fundamentala in programare. Putem sorta orice: numere, texte, orice are un criteriu de sortare. Pentru usurinta vom volosi numere in exemplele urmatoare.

Exista mai multe tipuri de sortari, precum:

  • Selection Sort
  • Insertion Sort
  • Merge Sort
  • Heap Sort
  • Bubble Sort
  • Quick Sort

In aceasta sectiune vom prezenta algoritmul Selection Sort (sortarea prin selectie)

Sortarea prin selectie (Selection Sort) face parte din algoritmii de sortare cu complexitate O(n^2), lucru care il face ineficient pentru sortarea multimilor mari.

Ideea algoritmului este simpla. Aducem cel mai mic element mereu in fata vectorului. Pentru a gasi mereu cel mai mic element ramas, trebuie sa parcurgem multimea de mai multe ori. Multimea este impartita in 2 submultimi imaginare: una sortata, care initial nu contine niciun element, si una nesortata, care contine toata multimea initiala

La fiecare pas, algoritmul gaseste elementul minim din multimea nesortata, apoi il introduce la finalul multimii sortate.

Atunci cand multimea nesortata nu mai are niciun element, algoritmul se opreste

Selection Sort video/animation soon

Sortarea se opreste atunci cand submultimea nesortata nu mai contine niciun element.

Asa cum stim, la fiecare pas, numarul de elemente nesortate scade cu 1.

Deci, algoritmul de sortare prin selectie va parcurge n pasi (n fiind numarul de elemente din multimea initiala). Fiecare pas presupune gasirea elementului minim din submultimea nesortata, adica parcurgerea unei submultimi de n, n-1, n-2, ..., 1 elemente. Indiferent de numarul de elemente din multime, acest algoritm executa (n-1) + (n-2) + ... + 3 + 2 + 1, adica N * (N-1)/2 comparari.

Deci, complexitatea algoritmului poate fi vazuta ca O(n^2).

Pseudocodul algoritmului este urmatorul:

for i ← 1 to n-1
    min ← i
    for j ← i+1 to n
      if A[j] < A[min] then
        min ← j
    end for
    if min != i then
      interchange A[i] and A[min]
  end for

Vom prezenta cateva implementari ale algoritmului:


function selectionSort(inputArr) { 
  let n = inputArr.length;
      
  for(let i = 0; i < n; i++) {
      let min = i;
      for(let j = 0; j < n; j++){
          if(inputArr[j] < inputArr[min]) {
              min=j; 
          }
        }
        if (min != i) {
            let tmp = inputArr[i]; 
            inputArr[i] = inputArr[min];
            inputArr[min] = tmp;      
      }
  }
  return inputArr;
}
                

public void sort(int arr[]) 
{ 
    int n = arr.length; 

    for (int i = 0; i < n-1; i++) 
    { 
        int min_idx = i; 
        for (int j = i+1; j < n; j++) 
            if (arr[j] < arr[min_idx]) 
                min_idx = j; 

        if (min_idx != i) {
          int temp = arr[min_idx]; 
          arr[min_idx] = arr[i]; 
          arr[i] = temp;
        } 
    } 
}
                

void swap(int *xp, int *yp) {  
    int temp = *xp;  
    *xp = *yp;  
    *yp = temp;  
}  
  
void selectionSort(int arr[], int n) {  
    int i, j, min_idx;  
  
    for (i = 0; i < n-1; i++)  
    {  
        min_idx = i;  
        for (j = i+1; j < n; j++)  
        if (arr[j] < arr[min_idx])  
            min_idx = j;  
  
        if (min_idx != i) swap(&arr[min_idx], &arr[i]);  
    }  
}
                

def selection_sort(L):
for i in range(len(L)-1):
    min_index = i
    
    for j in range(i+1, len(L)-1):
        if L[j] < L[min_index]:
            min_index = j
    
    if min_index != i:
        L[i], L[min_index] = L[min_index], L[i]
